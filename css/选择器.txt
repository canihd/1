}			\
a:visited{			   颜色只能存在一个生效↓以后↑不再生效
已访问的链接	color:	/
}
a:hover{
鼠标悬停在链接上时 color:
}
a:active{
选定的链接  color:
}
必须按照这个顺序写顺序乱则全都不生效
链接需要单独指定样式通用样式对链接不生效
